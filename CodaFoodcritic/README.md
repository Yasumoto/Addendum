# CodaFoodcritic

1. Binary is assumed to be at `/usr/local/bin/foodcritic`
2. Build the `CodaFoodcritic` target
3. Locate your `DerivedData` output directory
4. Place the `Addendum.framework` file in `~/Library/Frameworks`
5. Place `CodaFoodcritic.codaplugin` in `~/Library/Application\ Support/Coda\ 2/Plug-ins`
